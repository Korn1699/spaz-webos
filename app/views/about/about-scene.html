About
You are currently running a preview release of Spaz for webOS, an open source Twitter client. We think this version is pretty darn good, but we have more planned for 1.0, including:
  • Ping.FM integration
  • Uploading photos and images
  • Integration with Synergy
  • Search history & saved searches
Our plan is to charge $3 for the 1.0 version. This money will be used to cover hosting costs and compensate our contributors when possible.

Spaz is open source software with a New BSD-style license. We need volunteers to participate in the development process. If you're interested in helping, please join the Spaz Mobile Google Group.
